## 项目简介

基于`SpringBoot` + `MyBatis`（代码自动生成工具） + `Thymeleaf` 快速开发Web管理后台和Restful API项目原型

- 整合登录；
- 整合用户管理，团队管理和角色分配；
- 整合基于JQuery的常用UI组件库 + 深色统一风格布局；

### 运行项目

基于`maven`，通过`Application.java`直接`Run`

命令行运行：`java -jar simple.jar`

后台命令行运行：`nohup java -jar simple.jar &`
